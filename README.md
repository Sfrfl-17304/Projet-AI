# Our Project

This project is an AI-powered career guidance platform that analyzes user skills from resumes or questionnaires, matches them with suitable career paths, and provides personalized learning roadmaps. It uses AI-driven skill extraction, semantic matching, and knowledge graph technologies to support both beginners and experienced users.

## Features
- Skill extraction from CV or user input
- Career path recommendations
- Personalized skill gap analysis
- Time-based learning roadmaps
- Interactive knowledge graph visualization
- Chat-based career assistance

## Technologies
- AI / NLP for skill extraction and matching
- MongoDB for user profiles and session storage
- Neo4j for knowledge graph management
- Backend APIs integrated with frontend UI
