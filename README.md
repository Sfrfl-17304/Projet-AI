Our Project
This project is an AI-powered career guidance platform that helps users identify suitable career paths by analyzing their skills from resumes or questionnaires. It offers personalized learning roadmaps to address skill gaps using AI-driven skill extraction, semantic matching, and knowledge graph technologies. The system supports both beginners and experienced users through a modular backend and an interactive web interface.

Features
Skill extraction from CV or user input

Career path recommendations

Personalized skill gap analysis

Time-aware learning roadmaps

Interactive knowledge graph visualization

Chat-based assistance for career questions

Technologies
AI / NLP for skill extraction and matching

MongoDB for user profiles and session storage

Optional Neo4j for knowledge graph management

Backend APIs and frontend UI integration
